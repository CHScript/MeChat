# MeChat-实时聊天室

## 项目所用技术：
前端：vue vue-router vuex naive-ui axios socket.io-client  
后端：express redis mongodb socket.io  

## 项目启动命令：
```
cd client && npm install && npm run serve
cd serve && npm install && node app.js
```

## 具体实现功能：
+ 个人信息展示
+ 实时展示在线用户列表
+ 聊天记录数据的发送与接收
+ 登录/注册/修改密码/邮箱验证
+ 防抖函数控制发送按钮，防止用户多次提交表单
+ 用户登录或退出，都会动态更新在线用户列表
